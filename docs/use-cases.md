# Use Cases

## 1. Sales Order Query and Action

User requests sales order details and triggers follow-up actions such as notifications.

## 2. Email Automation

System generates and sends contextual emails using Outlook based on user requests.

## 3. Knowledge-Based Query (RAG)

User asks business-related questions and system retrieves answers from enterprise knowledge base.

## 4. Invoice Processing with Human-in-the-Loop

System extracts invoice data, validates it, and requests approval before processing.

## 5. Incident Creation and Management

System creates incidents in external systems based on issues detected or user requests.
